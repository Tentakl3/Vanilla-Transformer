# Vanilla Transformer (GPT + ViT)

## 🌐 Language / Idioma

[English](#english) | [Español](#español)

## English

This repository provides a minimal and educational implementation of Transformer-based architectures, including:

- A GPT-style autoregressive Transformer
- A Vision Transformer (ViT) built on top of the same codebase

The implementation is based on Andrej Karpathy's work in "Let's build GPT from scratch" and extends it to demonstrate how the same core ideas generalize beyond language to vision.

---

### Motivation

The goal of this repository is to present a clean, minimal, and transparent implementation of modern Transformer architectures.

Rather than relying on large frameworks, this project focuses on:

- Understanding the core building blocks of Transformers
- Demonstrating how GPT and ViT share the same underlying structure
- Providing a compact codebase that is easy to read, modify, and extend
